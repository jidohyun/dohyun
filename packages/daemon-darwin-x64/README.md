# @jidohyun/dohyun-daemon-darwin-x64

Pre-built dohyun daemon release for darwin-x64.

Optional dependency of [`@jidohyun/dohyun`](https://www.npmjs.com/package/@jidohyun/dohyun). npm picks the matching bundle automatically based on `os`/`cpu` — do not install this directly.

See the main package and <https://github.com/jidohyun/dohyun> for setup.
